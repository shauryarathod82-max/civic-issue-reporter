# civic-issue-reporter
 # Civic Issue Reporting System

This is a simple Java project where users can report and manage everyday civic issues like potholes, garbage, or water leakage. You can add issues, view them, and mark them as resolved.

I built this project to practice Java, OOP concepts, and basic data structures like ArrayList while working on something practical and real-world.


## Features

* Add new issues
* View all issues
* Mark issues as resolved

## Tech Used

* Java
* OOP Concepts
* ArrayList

## Future Improvements

* Add a simple GUI
* Connect with a database
* Add user login system
